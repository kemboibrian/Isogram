# Isogram

#### Date, 2024/04/23

#### By *Brian Kemboi*

## Description
This is a javaScript solution for An isogram - a word that has no repeating letters, consecutive or non-consecutive. Implement a function that determines whether a string that contains only letters is an isogram. Assume the empty string is an isogram. Ignore the letter case.
